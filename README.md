## Getting Started

First, add you RIOT API Key into a .env.local file.
 - make a copy of env.example
 - rename it to .env.local
 - change the value of RIOT_KEY variable to your key

Then you can run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
